# Stock Manager (Storekeeper)

> You handle all stock movements and ensure components are available and allocated before production begins.

{% hint style="info" %}
Roles in Odoo: **Storekeeper**. See Roles and Privileges for full access details.
{% endhint %}

### Your Tasks

#### Stock Allocation (after MRP is confirmed)

1. Access the pending materials request — via SU form or Inventory app
2. Check component availability — click Check Availability
3. Process the stock move — AutoFill and Validate, or update line by line
4. Handle backorders if needed — create a backorder or select No Backorder

#### Optional

* Process a recasting component request
