# P\&O (Prosthetist & Orthotist)

> You manage the full device lifecycle — from the initial request to the final delivery to the Service User.

{% hint style="info" %}
Roles in Odoo: **P\&O**. See Roles and Privileges for full access details.
{% endhint %}

### Your Tasks

#### Manufacturing a Device

1. Create the BoM Configuration — define side, size, color for the Service User
2. Create the MRP Order — initiate the manufacturing order
3. Process Work Orders — progress each step in the workshop
4. Validate the Manufacturing Order — mark production as done
5. Deliver the Device to the SU — complete the delivery

#### Other Materials

* Request materials for non-manufactured items — bandages, consumables, etc.
