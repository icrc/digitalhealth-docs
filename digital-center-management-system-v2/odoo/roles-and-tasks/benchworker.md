# Benchworker

> You carry out the physical manufacturing steps in the workshop.

{% hint style="info" %}
Roles in Odoo: **Benchworker**. See Roles and Privileges for full access details.
{% endhint %}

### Your Tasks

#### Daily Workshop Tasks

1. Find and open your assigned Work Orders — via the Manufacturing app or SU form
2. Start, pause, and complete each Work Order step — track time per step
3. Request Greenlight validation when required — click Request Validation and wait for approval

#### Optional

* Request new components for recasting — duplicate a picking or create a new request
