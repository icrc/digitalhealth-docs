# DCMS Administrator

> You manage the system configuration and ensure the platform is correctly set up for all other roles to work efficiently.

{% hint style="warning" %}
These tasks require broad administrative access. Always test configuration changes carefully before applying them in a live environment.
{% endhint %}

### Your Tasks

#### Product & Catalog Setup

1. Understand products and variants — key concepts before creating anything
2. Manage product categories — structure your product catalog correctly
3. Create a new product — including attributes and variants
4. Understand product attributes — required for correct BoM configuration
5. Archive a product — safely remove outdated products

#### Manufacturing Configuration

6. Create a new BoM Configuration Template — define components and operations for a product
7. Approve a BoM Template — make it available for P\&O to use

#### Stock Configuration

8. Understand picking/operation types — configure how stock moves are handled
9. Manage stock locations — set up warehouse, workshop, and other locations
10. Set up automatic replenishment — automate reorder rules

#### Troubleshooting & Support

11. Odoo Troubleshooting guide — quick fixes for common issues across all modules
