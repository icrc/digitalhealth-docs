# Request for Quotation (RFQ)

{% hint style="info" %}
## Roles recommended :  P\&O, Benchworker
{% endhint %}

## **🧭** Purchase Order Creation



## 🔄 Step-by-Step Flow&#x20;

### Step 1: Access the Purchase Module

Open the main dashboard and select the Purchase icon. You will see the list of existing RFQs and Purchase Orders.

<figure><img src="../../.gitbook/assets/image (361).png" alt=""><figcaption></figcaption></figure>

### Step 2: Create a New Request for Quotation (RFQ)

1. Click the New button in the top-left corner.
2. Select Vendor: In the "Vendor" field, start typing the name of your supplier (e.g. Sourcing Solutions) and select it from the list.
3. Order Deadline: Set the expected date by which the order should be confirmed.

<figure><img src="../../.gitbook/assets/image (362).png" alt=""><figcaption></figcaption></figure>

<figure><img src="../../.gitbook/assets/image (363).png" alt=""><figcaption></figcaption></figure>

### Step 3: Add Products to the Order

1. Under the Products tab, click "Add a product". Select the item from the dropdown menu (e.g. Plaster of Paris or Bandage).
2. Quantity: Enter the specific amount you wish to order in the "Quantity" column.
3. Unit Price: Ensure the price is correct. The system will automatically calculate the Taxes and Total at the bottom of the page.

<figure><img src="../../.gitbook/assets/image (367).png" alt=""><figcaption></figcaption></figure>

### Step 4: Save and Review

1. Click the Save button at the top-right corner to store the draft.
2. Review all details (Vendor, Products, Quantities, and Dates) to ensure accuracy.

<figure><img src="../../.gitbook/assets/image (366).png" alt=""><figcaption></figcaption></figure>

### Step 5: Confirm the Purchase Order

1. Once the details are verified, click the Confirm Order button.
2. The status of the document in the top right will change from RFQ to Purchase Order. The order is now official and recorded in the system.

<figure><img src="../../.gitbook/assets/image (368).png" alt=""><figcaption></figcaption></figure>

{% hint style="info" %}
Draft Status: While the document is in RFQ status, you can still edit any field.
{% endhint %}

{% hint style="info" %}
Totals: Always check the **Untaxed Amount** and **Total** at the bottom right before confirming to catch any pricing errors.
{% endhint %}
